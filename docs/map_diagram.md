# Brindlewick — Town Map & Spatial Layout

## Compass Orientation

- **North** = toward Copper Hill / mountains / trailhead  
- **South** = toward wilderness, farms, and the lake's far end  
- **East** = toward Lake Mirrowell  
- **West** = toward Maple Row (residential) and open country
